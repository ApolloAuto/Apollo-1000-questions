目前Apollo支持的硬件设备有哪些？
================================

目前Apollo支持的硬件设备有哪些？例如车辆底盘、计算单元、传感器等。

回答
----

Apollo对多种自动驾驶硬件设备提供了全面支持，以方便客户进行自动驾驶硬件选型，详细的硬件支持列表，请参考官方\ `硬件开发平台 <https://apollo.baidu.com/community/hardware>`__\ 页面。

如果您是硬件设备厂商，也欢迎你加入Apollo硬件生态，共同推动自动驾驶技术的发展。
